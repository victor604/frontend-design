# Changelog

All notable changes to the Frontend Design skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-27

### Added
- Initial release of Frontend Design skill
- Design thinking framework with 15+ aesthetic directions
- Typography mastery guide with font psychology
- Animation patterns library (CSS + Framer Motion)
- Color theory fundamentals
- Layout composition basics
- Screenshot analysis capability
- Production-ready code generation
- Mobile-first responsive patterns
- Accessibility guidelines (WCAG AA)
- Complete transformation examples
- Neo-Brutalist, Editorial, and Glassmorphism patterns

### Core Features
- Bold aesthetic directions (no generic AI patterns)
- Distinctive typography (avoiding Inter/Roboto/Arial)
- Context-aware design recommendations
- One memorable feature per design
- Production-grade implementation
- Comprehensive reference materials

### Documentation
- Main SKILL.md with complete design framework
- typography-mastery.md reference
- animation-patterns.md reference
- README with usage examples
- MIT License
- Contributing guidelines

### Supported Platforms
- Claude.ai
- Claude Code
- Cursor
- OpenCode
- Any Agent Skills-compatible assistant
